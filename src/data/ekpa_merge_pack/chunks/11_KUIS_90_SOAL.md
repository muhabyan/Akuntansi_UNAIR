# 90 Soal Latihan Pra UAS

Kuis Interaktif

## 90 Soal Latihan Pra UAS

Soal bersifat latihan konseptual. Klik jawaban, baca pembahasan, lalu lanjutkan soal berikutnya.

1. #### 1. [TM8] Etika berbeda dari moral karena etika terutama bersifat...

   1. emosional
   2. **reflektif dan argumentatif** ← jawaban benar
   3. administratif
   4. finansial

   **Pembahasan:** Etika menuntut alasan moral yang dapat diuji secara rasional.
2. #### 2. [TM8] Dalam etika bisnis Islam, konsep tanggung jawab menekankan bahwa...

   1. keuntungan adalah satu-satunya tujuan
   2. **setiap tindakan bisnis harus dipertanggungjawabkan** ← jawaban benar
   3. pasar bebas tanpa batas
   4. hukum positif tidak penting

   **Pembahasan:** Tanggung jawab mencakup dimensi sosial, hukum, profesional, dan spiritual.
3. #### 3. [TM8] Kasus korupsi pengadaan kitab suci menunjukkan bahwa simbol agama...

   1. pasti menjamin etika
   2. **tidak otomatis menjamin perilaku etis** ← jawaban benar
   3. tidak relevan sama sekali
   4. menghapus kebutuhan kontrol

   **Pembahasan:** Etika tetap membutuhkan integritas, kontrol, dan akuntabilitas.
4. #### 4. [TM8] Aksioma ihsan paling tepat dimaknai sebagai...

   1. minimal compliance
   2. **berbuat optimal dan baik** ← jawaban benar
   3. kebebasan tanpa batas
   4. dominasi pasar

   **Pembahasan:** Ihsan adalah kualitas melakukan kebaikan secara optimal.
5. #### 5. [TM8] Akuntan yang menolak invoice fiktif menunjukkan nilai...

   1. **amanah dan integritas** ← jawaban benar
   2. arogansi
   3. CEO duality
   4. income inequality

   **Pembahasan:** Penolakan fraud mencerminkan amanah, integritas, dan tanggung jawab.
6. #### 6. [TM9] Inclusion berarti organisasi...

   1. hanya merekrut orang seragam
   2. **menerima perbedaan dan memberi ruang partisipasi** ← jawaban benar
   3. menghapus semua aturan kerja
   4. menilai pekerja dari identitas

   **Pembahasan:** Inklusi berarti perbedaan diakui dan setiap orang dapat berkontribusi.
7. #### 7. [TM9] Reasonable accommodation paling dekat dengan...

   1. **penyesuaian wajar untuk kebutuhan tertentu** ← jawaban benar
   2. bonus manajemen
   3. pengurangan laporan
   4. larangan beribadah

   **Pembahasan:** Akomodasi wajar dapat berupa aksesibilitas, waktu ibadah, atau penyesuaian alat kerja.
8. #### 8. [TM9] Diskriminasi terjadi ketika keputusan kerja didasarkan pada...

   1. kompetensi relevan
   2. **identitas yang tidak relevan** ← jawaban benar
   3. hasil kerja terukur
   4. standar jabatan

   **Pembahasan:** Diskriminasi muncul saat identitas personal dipakai sebagai dasar perlakuan tidak adil.
9. #### 9. [TM9] Ketimpangan pendapatan menjadi isu etika karena terkait dengan...

   1. warna logo
   2. **keadilan dan legitimasi organisasi** ← jawaban benar
   3. jumlah aset tetap
   4. umur software

   **Pembahasan:** Kompensasi memengaruhi keadilan, motivasi, dan kepercayaan.
10. #### 10. [TM9] Kasus religious discrimination menuntut perusahaan menguji apakah kebijakan dress code...

    1. pasti benar
    2. **proporsional dan tidak diskriminatif** ← jawaban benar
    3. tidak perlu dijelaskan
    4. boleh melanggar hak

    **Pembahasan:** Dress code harus memiliki alasan bisnis yang sah dan tidak berlebihan.
11. #### 11. [TM10] Keahlian profesional akuntan modern mencakup...

    1. hanya debit-kredit
    2. **teknis, soft skill, dan etika** ← jawaban benar
    3. hanya hafalan standar
    4. hanya kemampuan bicara

    **Pembahasan:** Industri membutuhkan kombinasi kompetensi.
12. #### 12. [TM10] Contoh hard skill adalah...

    1. empati
    2. **audit dan perpajakan** ← jawaban benar
    3. kerja sama
    4. kepemimpinan

    **Pembahasan:** Audit dan pajak adalah keterampilan teknis.
13. #### 13. [TM10] Contoh soft skill adalah...

    1. PSAK
    2. rekonsiliasi bank
    3. **komunikasi profesional** ← jawaban benar
    4. jurnal penyesuaian

    **Pembahasan:** Komunikasi adalah soft skill inti.
14. #### 14. [TM10] Sikap profesional yang penting saat menemukan bukti janggal adalah...

    1. diam saja
    2. **skeptisisme profesional** ← jawaban benar
    3. menghapus data
    4. mengikuti vendor

    **Pembahasan:** Skeptisisme mendorong akuntan menguji bukti secara kritis.

    4. membuat jurnal

    4. alat promosi

    4. mahasiswa dan dosen

    4. komite audit hilang otomatis

    **Pembahasan:** CEO yang merangkap chair dapat mengurangi check and balance.
