# Cara Mendapat Nilai Tinggi

Rubrik Jawaban

## Cara Mendapat Nilai Tinggi

| Komponen | Jawaban Lemah | Jawaban Kuat |
| --- | --- | --- |
| Definisi | Definisi umum dan tidak presisi. | Definisi operasional, relevan dengan profesi akuntan. |
| Konsep | Menyebut istilah tanpa menjelaskan hubungan. | Menghubungkan konsep dengan kasus, standar, dan perilaku profesional. |
| Kasus | Menceritakan ulang kasus. | Menganalisis masalah etis, pihak terdampak, prinsip, risiko, dan rekomendasi. |
| Contoh | Contoh terlalu umum. | Contoh konkret: invoice fiktif, cut-off pendapatan, diskriminasi rekrutmen, CEO dominance, memo audit. |
| Bahasa | Bertele-tele dan tidak sistematis. | Bahasa formal, runtut, berbasis SPOK, dan langsung pada inti. |

**Formula jawaban cepat:** Definisi → konsep → kasus → prinsip etika → pihak terdampak → keputusan profesional → rekomendasi pengendalian.
